1. The fundamental spaces of linear algebra¶

1.1. Solution of linear systems¶

We can enconde RREF operations in a matrix by appending the identity to the matrix to which we apply the RREF.

\begin{equation} \begin{bmatrix} A & I \end{bmatrix} \rightarrow_{RREF} \begin{bmatrix} R & E \end{bmatrix} \end{equation}

Then clearly \(E\) is the matrix that takes \(A \to R\) (you can see this by premultiplication; only \(E\) could take \(I\) to \(E\)).

In the special case when \(A\) is square and invertible, this is how we computed the inverse matrix:

\begin{equation} \begin{bmatrix} A & I \end{bmatrix} \to_{RREF} \begin{bmatrix} I & A^{-1} \end{bmatrix} \end{equation}

1.2. Nullspace¶

The nullspace is an attribute of a linear operator, here a matrix \(A\). It is the set of all vectors for which \(A x = 0\).

\begin{equation} \mathcal{N}(A) = \left\{x \mid Ax = 0\right\}. \end{equation}

1.2.1. Finding a basis for the nullspace¶

I don’t know why, but I have a mental block about the procedure for finding a basis of the nullspace.

One way of understanding it is that we are analytically solving the system of equations that determins \(Ax = 0\) and expressing the solution set in terms of a set of spanning vectors.

I can remember how to do it with a mnemonic example:

Mnemonic example \(A = [1, 2, 3]\)

For \(A = [1, 2, 3]\), \([1, 2, 3]x = 0\) implies

\begin{align} x_1 + 2x_2 + 3x_3 = 0 \implies x_1 = -2x_2 - 3x_3 \end{align}

and in principle we are done. Any vector whose components follow this rule will sit in the nullspace of \(A\). But practically, we want a spanning set of vectors in the nullspace.

The idea is that we have expressed a constraint on \(x_1\), but $x_2 $ and \(x_3\) are completely unconstrained. So we introduce two dummy equations:

\begin{align} x_1 &= -2x_2 - 3x_3 \\ x_2 &= x_2 \\ x_3 &= x_3 \end{align}

Then writing the spanning set is a matter of grouping everything in terms of the unconstrained variables

\begin{equation} x= \begin{bmatrix} -2 \\ 1 \\ 0 \end{bmatrix} x_2 + \begin{bmatrix} -3 \\ 0 \\ 1 \end{bmatrix} x_3 \end{equation}

or that

\begin{equation} \mathcal{N}(A) = \text{span} \left\{ \begin{bmatrix} -2 \\ 1 \\ 0 \end{bmatrix}, \begin{bmatrix} -3 \\ 0 \\ 1 \end{bmatrix} \right\} \end{equation}

General procedure:

So practically, solving the system amounts to an RREF operation, and then we can read off the nullspace using the following shortcut

\begin{equation} RREF(A) = \begin{bmatrix} F \\ 0 \end{bmatrix} \qquad\to\mathcal{N}(A) = \begin{bmatrix} -F \\ I \end{bmatrix} \end{equation}

Where we express the pivot variables in terms of the free variables. Each free variable corresponds to a basis vector of the nullspace.

Of course, if we ever need this practically, just ask a libary to do it.

1.3. Column space¶

The columnspace of a linear operator \(A\) is the set of all linear combinations of the columns of \(A\), the set

\begin{equation} \mathcal{C}(A) = \left\{ \text{all vectors }Ax \right\} \end{equation}

It can be thought of as - All linear combinations of the columns of \(A\) - the range of \(A\), everything that \(A\) can “hit”

The equation \(Ax = b\) is solvable exactly when \(b \in \mathcal{C}(A)\).

1.4. Visual assessment of matrices¶

By visually inspecting a matrix \(A\in \mathbf{R}^{m\times n}\), we can say quite a bit about the nullspace, column space, and the solution set \(Ax = b\).

Inputs and outputs

\(A\in \mathbf{R}^{m\times n}\) takes as input a vector in \(x\in \mathbf{R}^{n}\) and outputs a vector \(Ax \in \mathbf{R}^{m}\). An abstract linear operator \(A(x)\colon \mathbf{R}^{n}\to \mathbf{R}^{m}\).

1.4.1. “Fat” matrices¶

A fat matrix (\(m < n\)) operates on a vector in \(\mathbf{R}^{n}\) and squashes it into a lower-dimensional vector. We assume that the fat matrix has full “row rank”, that \(RREF(A)\) has a pivot in each row.

Solutions to linear systems

The linear system will be consistent for any \(b\); in this case, there is a linear dependence in the columns, there must be infinite solutions.

Full “row” rank \(\implies\) linear system is consistent for any \(b\).

The nullspace \(\mathcal{N}(A)\) can not be trivial. \(Ax = 0\) will have infinitely many solutions due to the redundancy in columns.

1.4.2. “Tall” matrices¶

A tall matrix (\(m>n\)) operates on a vector in \(\mathbf{R}^{n}\) and returns a higher-dimensional vector. We assume that the fat matrix has full “column rank”; that \(RREF(A)\) has a pivot in each column.

Solutions to linear systems

Since the matrix has full column rank, if there is a solution for a right-hand side \(b\), it is unique. Full column rank means no free variables.

Full “column” rank \(\implies\) if there is a solution to \(Ax = b\), it’s unique.

The columnspace \(\mathcal{C}(A)\) can not span the whole space of \(\mathbf{R}^n\), so the system can not be consistent for all \(b\in \mathbf{R}^m\).

1.4.3. Square matrices¶

A square matrix (\(m=n\)) operates on a vector in \(\mathbf{R}^{n}\) and returns a vector in the same space.

Full rank \(\implies\) there is a unique solution \(x\) for every right-hand side \(b\).
