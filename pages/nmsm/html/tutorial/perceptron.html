6. Random linear classifiers¶

6.1. Hyperplanes¶

a hyperplane is a set of the form \(\{x\,|\, a^T x = b\}\) where \(a, x\in \mathbb{R}^n\) and where \(b\in \mathbb{R}\).

There are a few geometric interpretations of the hyperplane \(\{x\,|\, a^T x = b\}\) - if \(b=0\), then this is the set of all vectors that are orthogonal to \(a\); that is, the solution can be thought of as the plane defined by the normal vector \(a\). - if \(b\neq 0\), then the plane in the \(b=0\) case is offset from the origin. We can think of it like

\[\{x\,|\, a^Tx - b = 0\} = \{x \,|\, a^T(x - x_0) = 0\}\]

where \(x_0\) is any vector such that \(a^T x_0 = b\). This makes sense, because any element of the original offset hyperplane can be subtracted to recover the hyperplane passing through the origin.

The perpendicular offset of the origin \(d\) can be computed by noting that the vector \(\mathbf{0} + da\) lies on the hyperplane. Hence

\[a^T(0 + d a) = b \Rightarrow d = \frac{b}{a^T a}\]

if \(a^T x + b = 0\), then $a_1 x + a_2 y + b = 0 $ hence

\[y = -\frac{1}{a_2}(a_1 x +b )\]

6.2. signed distance¶

Let ppp be an arbitrary point in \(\mathbb{R}^d\). Give a formula for the signed perpendicular distance from the hyperplane specified by \(\theta, \theta_0\), to this point \(p\).

Let \(d\) be the signed distance we seek. Note that \((p - \theta d)\) is on the hyperplane. Hence \(\theta^T(p + \theta d) + \theta_0 = 0\) and we have

\[d = \frac{\theta^T p + \theta_0}{\theta^T \theta}\]
[4]:

def signed_dist(x, θ, b):
    return ( np.dot(θ, x) + b )/np.dot(θ,θ)

[5]:

def classify(x, θ, b):
    signs = np.sign(signed_dist(x, θ, b))
    return signs.astype(int)

Get the signed distance and classify a set of random points according to our hyperplane.

6.3. Creating a random linear classifier¶

In creating a linear classfier, we need a way to score the classifier. Here, we simply return the number of testing data that were correctly classified.

[8]:

def score(data, labels, θ, b):
    classifications = classify(data, θ, b)
    correct = (classifications == labels)
    return np.sum(correct)

[9]:

def best_separator(data, labels, Thetas, bs):
    """
    :param Thetas: shape (m, d) each row is a θ candidate
    :param bs: shape (m,) bs to go with each θ
    """
    scores = np.zeros_like(bs)
    for idx, (θ, b) in enumerate(zip(Thetas, bs)):
        scores[idx] = score(data, labels, θ, b)
    max_idx = np.argmax(scores)
    return Thetas[max_idx], bs[max_idx], scores, max_idx

[10]:

def generate_random_separators(d, m, b_range=10):
    """
    :param d: dimension of problem
    :param m: number of separators returned
    :returns Theta: (m, d) matrix of random separators
    :returns bs: (m,) vectors of offsets
    """
    Theta = np.random.random((m, d)) - 0.5 # random vectors around origin
    bs = np.random.random(m) * b_range
    return Theta, bs

Now we are ready to randomly generate a linear classifier. We will see if we can generate something that matches the classifier above.
