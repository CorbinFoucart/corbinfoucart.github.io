5. Duality¶

import numpy as np
import matplotlib.pyplot as plt

5.1. The Lagrangian¶

Suppose we have a standard optimization problem, which may not be convex.

\begin{align} \text{minimize} \; & f_0(x) & \\ \text{subject to}\; & f_i(x)\leq 0& i=1,\ldots,m \\ & h_i(x) = 0& i=1,\ldots,p \end{align}

Where \(x\in \mathbf{R}^{n}\). We assume the domain \(\mathcal{D} = \bigcap_{i=1}^m \mathbf{dom} f_i \cap \bigcap_{i=1}^p \mathbf{dom} h_i\) is nonempty.

The Lagrangian \(L\colon (\mathbf{R}^{n}\times \mathbf{R}^{m}\times \mathbf{R}^{p})\to R\) associated with the problem as

\begin{equation} L(x,\lambda,\nu) = f_0(x) + \sum_{i=1}^{m} \lambda_i f_i(x) + \sum_{i=1}^{p} \nu_i h_i(x), \end{equation}

where the idea is to express the constraints by augmenting the objective function with a weighted sum of them. The vectors \(\lambda\) and \(\nu\) are called the dual variables or Lagrange multiplier vectors associated with the original problem.

This has an interesting interpretation. When the problem is in objective function + constraint form, we are completely agnostic to “how well” the constraints are satisfied, as long as the point is feasible. If there are multiple points in \(X_{opt}\), they are all equally optimal.

With the Lagrangian, we remove the idea of feasibility. Now, if you want to violate a constraint, you pay for it with price \(\lambda_i\) or \(\nu_i\) (Stephen Boyd calls these “irritation functions”). But conversely, if you come in “under budget” on a constraint, you get a subsidy \(\lambda_i\) or \(\nu_i\). In that sense, the Lagrangian cares not only about how small you can make the objective function, but how well you do on the constraints. In that case, we do care about how well the constraints are satisfied.

If we wanted the Lagrangian formulation to reject any unfeasible points, we simply choose the \(\lambda_i\) for example to take a value of \(\infty\) outside of the feasible set to express “infinite displeasure” with violating the constraints.

5.2. The Lagrange dual function¶

The Lagrange dual function as the minimum value of the Lagrangian over \(x\). For \(\lambda\in \mathbf{R}^{m}\), \(\nu \in \mathbf{R}^{p}\),

\begin{equation} g(\lambda,\nu) = \inf_{x\in D} L(x,\lambda,\nu) = \inf_{x\in D} \left( f_0(x) + \sum_{i=1}^{m} \lambda_i f_i(x) + \sum_{i=1}^{p} \nu_i h_i(x), \right) \end{equation}

The dual function bounds the optimal value \(p^*\) below on the feasible set. If \(x\) is not feasible, then the contraints add to the Lagrangian and the value could clearly exceed \(f_0(x)\).

We give some 1D examples for intuition.

Single inequality constraint

[2]:

f0 = lambda x: 2 + np.exp(-x)*np.sin(np.pi*x)
f1 = lambda x: x**2 - 1
L = lambda x, λ: f0(x) + λ*f1(x)

[3]:

x = np.linspace(-2, 2, 200)

# where the feasible set is
where_feas = f1(x) <= 0
where_infeas = f1(x) > 0

x_feas = x[where_feas]
x_infeas = x[where_infeas]

x_star = x_feas[np.argmin(f0(x_feas))]
p_star = np.min(f0(x_feas))

fig, ax = plt.subplots(1,1, figsize=(5,5))
ax.plot(x, f0(x), label=r'$f_0(x)$')

λs = np.linspace(0,1,5)
for λ in λs:
    ax.plot(x, L(x, λ), ':', color='gray')

ax.scatter(x=x_star, y=p_star, c='k', label=r'$p^*$')

ymin, ymax = ax.viewLim.y0, ax.viewLim.y1
ax.fill_between(x=[-1,1], y1=[ymin,ymin], y2=[ymax, ymax],
                color='grey', alpha=0.1,
                label='feasible set')
plt.legend(bbox_to_anchor=(1.4,1))
plt.show()

The dual function \(g(\lambda)\) is a function of \(\lambda > 0\) only.

[6]:

# compute the dual function
λs = np.linspace(0,1,50)
g = np.zeros_like(λs)
for idx, λ in enumerate(λs):
    g[idx] = np.min(L(x, λ))

plt.plot(λs, g, label=r'$g(\lambda)$')
plt.plot(λs, np.ones_like(λs)*p_star, 'k--', label='p^*')
plt.xlabel(r'$\lambda$')
plt.legend()
plt.show()
