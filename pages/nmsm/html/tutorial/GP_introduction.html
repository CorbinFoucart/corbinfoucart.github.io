1. Introduction¶

A Gaussian process is a distribution over functions \(f(x)\). The distribution is defined by a mean function \(\mu(x)\) and a positive definite covariance function \(k(x,x^\prime)\) (sometimes called a covariance kernel).

\begin{equation} f(x) \sim \mathcal{GP}\left(\mu(x),\, k(x,x^\prime)\right) \end{equation}

Sampling from a Gaussian like \(\mathcal{N}(0,1)\) returns a value as a sample; sampling from a Gaussian process returns a function \(f(x)\).

Gaussian processes have the property that for every finite subset \(X=\left\{x_1,\ldots,x_n\right\}\) of the domain, the marginal distribution is a multivariate Gaussian distribution

\begin{equation} f(X) \sim \mathcal{N}\left(\mu(X), k(X,X)\right) \end{equation}

where \(\mu(X)\in\mathbf{R}^{n}\) and \(k(X,X)=\Sigma\in\mathbf{R}^{n\times n}\) is the covariance matrix. While \(f(X)\) is a finite-dimensional distribution, the Gaussian process \(\mathcal{GP}(\mu(x),\, k(x,x^\prime))\) has a continuous input domain and is infinite dimensional.

The point of this notebook will be to introduce the basics of GP’s that will be revisted in much more detail in later notebooks.

1.1. Sampling from Gaussian Processes¶

The covariance kernel \(k(x,x^\prime)\) is the crucial ingredient in the behavior of the samples \(f(x)\) drawn from the Gaussian process; it is the object that encodes the assumptions about the functions which we wish to draw from the distribution. We’ll go into much more detail about covariance kernels in later notebooks; we give only the basic ideas here.

Discretely sampling from a Gaussian process (that is, generating \(f(X)\) at points \(X\)) is only a matter of sampling from the multivariate distribution \(\mathcal{N}(\mu(X),\Sigma)\). We consider only two common covariance kernels, which we will visit in more generality later.

The exponentiated quadratic covariance kernel \begin{equation} k\left(x, x^\prime\right) = \sigma^2\exp\left(-\frac{1}{2 \ell^2} \left\lVert x - x^\prime\right\rVert_2^2 \right) \end{equation}

[5]:

def exponential_quadratic_kernel(xa, xb, σ2=1, ℓ=1):
    """xa, xb are (n,) vectors of spatial locations"""
    L2_distance = spatial.distance.cdist(xa[:,np.newaxis],
                                         xb[:,np.newaxis], 'minkowski', p=2)
    return σ2*np.exp(-1/(2*(ℓ**2))* L2_distance**2)

The 1D Brownian motion covariance kernel \begin{equation} k\left(x,\, x^\prime\right) = \min\left(x, x^\prime\right) \end{equation}

This is the covariance kernel associated with a so-called Wiener process.

[6]:

def brownian_kernel(xa, xb):
    # Brownian motion only defined on [0,\infty)
    assert np.min(xa) >= 0
    N = len(xa)
    Σ = np.zeros((N,N))
    for i in range(N):
        for j in range(N):
            Σ[i,j] = np.minimum(xa[i], xb[j])
    return Σ

Realizations of each process look very different depending on the covariance kernel.

[10]:

dist = np.zeros((N_pairwise, N_pairwise))
for i  in range(N_pairwise):
    for j in range(N_pairwise):
        x, y = X[i], Y[i]
        xprime, yprime = X[j], Y[j]
        sq_dist = (x-xprime)**2 + (y-yprime)**2
        dist[i,j] = sq_dist

ℓ=0.02
Σ = np.exp(-1/(2*ℓ)*dist)

n_real = 2
random_field = np.random.multivariate_normal(
    mean=np.zeros(N_pairwise), cov=Σ, size=n_real)
